# Hermes Desk

An ambient AI agent terminal built on the M5Stack Tab5.

Powered by Hermes Agent. The agent speaks to you.

## Features
- Voice Conversation
- Proactive Notifications
- Passive Dashboard
- Active Visuals

## Hardware
- M5Stack Tab5 (ESP32-P4, 5-inch IPS 1024x600)
- Host machine (Linux, same LAN)

## Milestones
- M1: Voice Terminal (In Progress)
- M2: Proactive Notifications (Planned)
- M3: Passive Display (Planned)
- M4: Active Display (Planned)
